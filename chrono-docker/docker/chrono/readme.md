# OptiX Install

You will need to save the OptiX 9.0 build file here in order for Chrono Sensor to build and run properly on hosts that support OptiX 9.0. The file should have exactly the same name as when downloaded, which should be:

```
NVIDIA-OptiX-SDK-9.0.0-linux64-x86_64.sh
```

You can download OptiX from the NVIDIA DesignWorks OptiX page and pick the 9.0.0 installer that matches the filename above.
